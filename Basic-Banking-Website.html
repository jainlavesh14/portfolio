  • Github URL: Basic Banking Web App

Description

The project is a banking website implemented using Django framework and integrated with Google API for robot validation. It consists of five web pages that work together to provide various banking services.

The Add Customer page allows users to add new customers to the bank's database by entering details such as name, email, mobile number, account type, starting balance, and residential address. The "View Customer" page displays a list of all current customers along with their updated details. The Transfer Money page enables users to transfer funds by selecting a sender and beneficiary. After validating the transaction, the sender's account is debited, and the receiver's account is credited. Both parties involved in the transaction receive email alerts about the transfer. The Transaction History page allows users to view the transaction history of a specific account holder. Users can also receive their transaction history as a PDF attachment via email. Additionally, the project includes an About page for information purposes and a Contact/Reach Us page with contact details and social media links for users to get in touch. Overall, this banking website offers essential functionalities and seamless integration to provide a comprehensive online banking experience.

In conclusion, the implementation of this banking website using Django has successfully provided a user-friendly platform for various banking services. The project demonstrates the ability to seamlessly integrate multiple web pages to create a cohesive banking experience.
