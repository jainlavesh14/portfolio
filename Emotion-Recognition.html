Description

The Emotion Recognition project aims to predict the emotions of multiple users captured in a live camera frame based on their facial expressions. The system supports seven types of emotions: Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise. The training set consists of 28,709 examples images and the testing set consists of 3,589 examples images. The dataset can be found on Kaggle at the following link: https://www.kaggle.com/msambare/fer2013.

The Emotion Recognition system relies on machine learning techniques to analyze and classify facial expressions. It is likely built using deep learning algorithm such as Artificial Neural Network (ANN) and Convolutional Neural Networks (CNNs), which have been successful in image recognition tasks. Deep learning models are trained on the provided dataset to learn patterns and features associated with different emotions.

Tensorflow and Keras libraries are used. TensorFlow is an open-source machine learning framework developed by Google. It provides a comprehensive set of tools and libraries for building and deploying machine learning models, including deep learning models. TensorFlow is widely used in the research and industry communities for various tasks, including image recognition, natural language processing, and more. Keras, on the other hand, is a high-level neural network library that runs on top of TensorFlow (among other backend options). It provides a user-friendly and intuitive interface for designing, training, and evaluating neural network models. Keras simplifies the process of building and experimenting with different architectures and makes it easier to implement deep learning models, including Artificial Neural Networks (ANNs) and Convolutional Neural Networks (CNNs).

Overall, the Emotion Recognition project leverages a dataset of facial images to train a machine learning model capable of predicting emotions based on live camera input. It utilizes techniques from deep learning and potentially employs convolutional neural networks to accomplish this task.
